Project Description: Tech.IT

With the use of third-party APIs, this application was created to give Software Developers the ability to read the latest technology industry news so they can keep up-to-date with changes in the industry. This application also gives Software Developers the ability to lookup available jobs in the field so they can advance their professional careers. The application also features a section where inspirational quotes from technology leaders are displated to keep Developers motivated.

This is a simple calendar application that allows the user to save events for each hour of the day. This app will run in the browser and feature dynamically updated HTML and CSS powered by jQuery. Moment.js library is in use to work with date and time.
